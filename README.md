### Requirements

1. C/C++ compiler
2. CMake & Make

### How to start?

1. Clone the repo

```sh
git clone https://github.com/budchirp/cpp-template && cd cpp-template
```

2. Change project name in `CMakeLists.txt`

3. Compile the code

```sh
mkdir build && cd build
cmake .. && make -j$(nproc)
```

> NOTE: If you're using fish, remove the `$` sign.

4. Have fun!

```sh
./PROJECT_NAME
```
